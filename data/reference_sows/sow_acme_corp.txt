STATEMENT OF WORK
Client: Acme Corporation
Contract Date: 2025-11-15
Project: Cloud Environment Provisioning

1. SCOPE OF WORK
Acme Corporation requires a cloud-hosted environment to support their internal CRM application.
The application serves approximately 200 concurrent users across 3 regional offices.

2. TECHNICAL REQUIREMENTS
- Compute: The application requires moderate compute capacity. The CRM application is
  Java-based and requires at least 4 vCPUs and 16GB RAM per application server.
  Two application servers are needed for redundancy.
- Storage: Total data footprint is estimated at 500GB. The database requires SSD-grade
  storage for acceptable query performance. Estimated IOPS requirement is 3000.
- Networking: Standard VPC setup with 2 subnets (one public, one private).
  A load balancer is required to distribute traffic across application servers.
  VPC CIDR: 10.1.0.0/16
- Services Required:
  * Cloud SQL (PostgreSQL) - primary database
  * Cloud Monitoring - standard monitoring
  * Cloud Logging - centralized logging

3. AVAILABILITY REQUIREMENTS
- High availability is required (99.9% uptime SLA)
- Daily backups with 30-day retention

4. USER ESTIMATE
- 200 concurrent users
- Expected to grow to 500 within 18 months
