STATEMENT OF WORK
Client: Nimbus Retail Solutions
Contract Date: 2026-02-20
Project: E-Commerce Platform Hosting

1. SCOPE OF WORK
Nimbus Retail Solutions requires a cloud environment to host their e-commerce platform.
The platform handles online ordering, inventory management, and payment processing
for a mid-sized retail chain with 25 store locations.
The system must handle seasonal traffic spikes during holiday periods.

2. TECHNICAL REQUIREMENTS
- Compute: The e-commerce application needs 4 vCPUs and 16GB RAM per server.
  Three application servers are needed to handle normal traffic with headroom
  for holiday spikes.
- Storage: Total storage is estimated at 750GB including product catalog,
  order history, and customer data. SSD storage is required for the transactional
  database. IOPS requirement is 5000.
- Networking: VPC with 3 subnets (web, application, database).
  External load balancer is required for customer-facing traffic.
  VPC CIDR: 10.5.0.0/16
- Services Required:
  * Cloud SQL (PostgreSQL) - order and inventory database
  * Cloud Storage - product images and static assets
  * Cloud CDN - content delivery for static assets
  * Cloud Monitoring - performance monitoring
  * Cloud Logging - transaction logging

3. AVAILABILITY REQUIREMENTS
- High availability required (e-commerce cannot have downtime during business hours)
- Daily backups with 60-day retention

4. USER ESTIMATE
- 500 concurrent users (up to 2000 during peak holiday)
